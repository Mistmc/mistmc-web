# 关于服务器安全性[必看]
:::caution 警告

- 以下是为了确保您的财产不遭受损失默认开启的！！！服务器默认开启，无需自行配置

:::
- ISeeYou录制
    ISeeYou是一个可以利用Replay API，以mcpr格式录制玩家的全部游玩过程的插件

    可以轻松的验证您是否使用了作弊软件

    可以轻松的录制您创建您基地的全过程，且您的设备不需要任何额外程序以及额外负载

    ### 管理的保证
    除非遇到您被人举报或需要收集优秀玩家素材，否则管理是不会观看您的游玩录像的

- co i保护（CoreProtect插件）
    CoreProtect是数据记录和防止恶意破坏的工具，在您未圈地导致的被其他玩家破坏后，可以请求管理手动回档部分区域。

    详情可查看: [co i使用指南](./guide/function/coi)
    ### 如果忘记圈地
    如果在领地外物品丢失，可以联系管理享受财产找回，半年一次（快快圈地！不要再麻烦管理啦！）
