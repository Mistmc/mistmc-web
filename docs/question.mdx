# 常见问题
## 问: 服务器会换档吗？
答: 我们保证，如果没有出现极大的问题，服务器将永远不会换档。
## 问: 我怎么进不去服务器了？
答: 一般情况下服务器都能够顺畅无阻地进入游玩。但个别情况下，您可能无法进入。

:::info 未知的主机、无法解析主机名、网络不可达、Unknow host、Network is unreachable...

1. 首先，你应检查 [服务器地址](https://mistmc.top/docs/start#%E5%8D%B3%E5%88%BB%E5%8A%A0%E5%85%A5%E6%B8%B8%E6%88%8F) 是否输入正确
2. 多刷新几次这种情况有可能是因为你随机到的DNS地址的穿透隧道出现了问题，刷新后就有几率定位到其他隧道以进入游戏
<Admonition type="tip">
    - 刷新服务器列表和加入服务器均会随机查询一次DNS地址，你只需要确定服务器列表中有几率连接到服务器，那么只需要多试几次就可以进入服务器
</Admonition>
3. 检查后如果输入正确，[清理 DNS 缓存](https://jingyan.baidu.com/article/59a015e34fcd70f7948865a9.html) 后使用原先的地址进入
4. 如果上一步没有用，尝试将 DNS 服务器设置为 [114DNS](https://www.114dns.com/faq.html#win7) 并 [清理 DNS 缓存](https://jingyan.baidu.com/article/59a015e34fcd70f7948865a9.html) 后使用原先的地址进入

如果都不行，那多半是网络穿透服务商全寄了的问题
当然也有可能在维护
