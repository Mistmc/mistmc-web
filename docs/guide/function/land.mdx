# 圈地
:::note 💴钱啊

- 本服务器斥巨资购买了lands插件的使用权。真的不容易啊，呜呜~
:::
本服务器所有圈地内容全部采用Lands插件，且一次圈地可以圈一整个区块！

<strong>/land menu</strong> 即可打开菜单

<strong>/land create name</strong> 即可创建领地，每个人可以创建10个领地.  name 可以自行填写

<strong>/land edit</strong> 即可把当前领地设为编辑目标.    #老万的提醒！ 在每次进行任何操作前都别忘了设置编辑目标！

<strong>/land claim auto</strong> 即可开始圈地模式，此时您走过的所有相邻区块都会被合并到当前编辑目标中！

<strong>/land setspawn</strong> 即可设置领地传送点

剩下的操作建议在land的菜单中完成，更加直观！
