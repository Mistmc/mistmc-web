# 开始
岚·域服务器是一个由玩家社区支持的良心商业服务器

[关于良心的特别说明](./goodxin)

服主为mc506lw，由Mist团队开发
## 游玩前的准备

服务器版本：1.20.4  
登录：正版或离线版

:::tip 与原版 Minecraft 不一样的行为

- 火势不可蔓延（仅限生存世界的荒野）
- 苦力怕不可炸毁方块（仅限生存世界）

:::

:::caution 以下内容已被启用，滥用将会被管理员封禁

- 活塞复制：包括 TNT 复制,地毯复制,线复制

:::
