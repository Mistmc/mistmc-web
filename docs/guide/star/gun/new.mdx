# 更新日志

## 1.0 枪械更新
### 更新日志 V1.0.0：手霰步更新
2024/3/24

- 新增内容：
    - 子弹：
        - 小口径子弹：手枪弹药，造价很便宜
        - 大口径子弹：步枪弹药，造价比较便宜
        - 霰弹：霰弹枪弹药，造价比较贵
        - 马格南子弹：马格南手枪弹药（尚未实装），造价全插件最贵
        - 火球：火球发射器弹药，造价贵
    - 手枪：
        - 黑尾：经典手枪，射速快，伤害比较低
        - Red9：比黑尾伤害高，但稍微笨重，很少有人会用
        - 祈福者：较重型的手枪，未升级之前威力逊色，需要历经雕琢才能闪耀光芒
    - 霰弹枪：
        - W870：标准霰弹枪
    - 步枪：
        - SE080：中距离战斗的好手，敏捷但是子弹不稳定（提示：佩戴南瓜瞄准镜可以降低子弹漂移）
        - SE_F：为狙击而生的步枪，享受远程攻击的快乐吧！
    - 火球发射器：
        - 火球发射器：火箭筒，大范围杀伤，但是会破坏地形
    - 升级包：
        - 手枪升级包、步枪升级包、霰弹枪升级包、火球发射器升级包：升级武器，通过抽奖取得
    - 材料：用于制作弹药，枪械及配件的东西
        - 压缩火药：9火药压缩而成，用于制作马格南弹药
        - 枪管：用于合成手枪
        - 粗枪管：用于合成较大型的枪
        - 简单机械模块：用于合成较简单的武器
        - 复杂机械模块：用于合成较复杂的武器
        - 撞针：用于合成武器
        - 螺纹栓：用于合成伤害高的武器（目前无用）
        - 手枪握把、步枪握把、霰弹枪握把、火球发射器握把：合成对应武器
        - 弹丸组：用于合成霰弹
    - 配件：
        - 粗放南瓜瞄准镜：降低步枪弹丸离散程度
