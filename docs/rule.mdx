# 服务器守则[必读]

``` info "名言"
    “法者，治之端也。”
                                            ——韩非子
```
规定是管理和治理的根本为了维护岚·域我的世界服务器的和谐与秩序，我们特此制定以下游戏规则。请所有玩家认真阅读并遵守。

## 尊重社区成员

我们致力于打造一个友好和尊重的游戏环境。以下行为是严格禁止的：

- **禁止歧视性言论**：请勿发表任何有关种族、性别、年龄等的歧视性言论。
- **禁止骚扰和欺凌**：请勿进行任何形式的骚扰、欺凌或人身攻击。
- **保护个人隐私**：请勿泄露他人的个人信息。
- **禁止刷屏**：请勿在聊天中重复发送相同或相似的信息。
- **禁止不当内容**：请勿发布不当的性内容或裸露图片。
- **禁止恶意内容**：请勿散布恶意软件、病毒或进行欺诈行为。
- **禁止虚假信息**：请勿散布虚假信息或进行诽谤。
- **禁止负面内容**：请勿发布任何鼓励伤害自己或他人的内容。

违反上述规定可能会导致<span style={{ color: 'red' }}>**禁言、监禁或封号**</span>等处罚。

## 维护服务器稳定

请不要进行过度消耗服务器资源的行为：

- **限制实体数量**：请勿囤积大量实体或建造可能导致服务器负载过重的装置。
- **合理规划节点**：请合理规划您的货运节点，避免同时运行大量节点。

## 保护他人的建筑

请尊重并保护他人的建筑劳动成果：

- **禁止破坏**：请勿破坏或改变他人的建筑物。
- **禁止损害**：请勿使用任何手段对建筑造成损害。

## 公平游戏

我们鼓励所有玩家公平竞技：

- **禁止作弊**：请勿修改游戏数据或利用漏洞。
- **禁止未授权软件**：请勿使用未授权的第三方软件干预游戏。

## 尊重玩家财产

请不要偷取或占用他人的物品：

- **禁止偷盗**：请勿未经许可拿取他人物品。
- **禁止占用**：请勿盗窃他人的宠物或物品。

## 禁止恶意伤害

请不要无故伤害其他玩家或其宠物：

- **禁止恶意攻击**：请勿故意设置陷阱或使用爆炸物伤害他人。
- **PVP需双方同意**：经双方同意的PVP除外。

## 维护良好的创作环境

我们鼓励创造性和自由表达，但不应包含不当内容：

- **禁止不适当的创作**：请勿创建不适当的像素画、书籍、用户名或装备名。

违反这些规定将会受到相应的处罚。

我们感谢您的理解与合作，让我们共同努力，维护岚·域我的世界服务器的秩序与和谐。祝您游戏愉快！
