# 传送

:::tip 建议

- 世界和传送点传送建议使用菜单

:::
## 玩家互传
使用<strong>/tpa 玩家名</strong> 可以向玩家发送请求，若对方同意则将会在3秒后把您传送到对方的位置

使用<strong>/tpahere 玩家名</strong> 可以向玩家发送请求，若对方同意则将会在3秒后把对方传送到您的位置

使用<strong>/tpaccept 则可接受传送请求</strong> ,/tpadent 可拒绝
## 世界/传送点传送
使用<strong>/warp name</strong> 可以传送到对应传送点和世界
